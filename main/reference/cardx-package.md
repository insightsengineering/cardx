# cardx: Extra Analysis Results Data Utilities

Create extra Analysis Results Data (ARD) summary objects. The package
supplements the simple ARD functions from the 'cards' package, exporting
functions to put statistical results in the ARD format. These objects
are used and re-used to construct summary tables, visualizations, and
written reports.
