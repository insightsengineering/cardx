# Extract data from wald.test object

Extract data from wald.test object

## Usage

``` r
.extract_wald_results(wald_test)
```

## Arguments

- wald_test:

  (`data.frame`)  
  wald test object object from
  [`aod::wald.test()`](https://rdrr.io/pkg/aod/man/wald.test.html)

## Value

a data frame containing the wald test results.
